== NAME

prob2lts-sym - Symbolic reachability analysis for the B-Method, through ProB.

== SYNOPSIS

*{manname}* [OPTION]... '/path/to/socket.probz' ['output.etf']

== DESCRIPTION

{manname} performs a reachability analysis on a specification provided
through '/path/to/socket.probz'.  'output.etf' specifies the name of the output
archive, written in manpage:etf[5] format.
